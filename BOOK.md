## 使用说明

本文档包含易经64卦的完整原文数据,每卦包含以下内容:

- **卦序与卦名**: 六十四卦的排序和名称
- **卦象**: 上卦、下卦组成及六爻阴阳
- **卦辞**: 易经原文对整卦的判断
- **象辞**: 《象传》对卦象的解释
- **爻辞**: 六爻各自的说明(初、二、三、四、五、上)
- **彖辞**: 《彖传》对卦义的阐述(部分卦)

## 上经(1-30卦)

### 第1卦 乾卦(乾为天) ䷀

**卦象组成**:
- 上卦:乾☰(天)
- 下卦:乾☰(天)
- 六爻:上九、九五、九四、九三、九二、初九(六爻皆阳)

**卦辞**:
```
乾:元,亨,利,贞。
```

**白话译文**:
乾卦:大吉大利,吉利的贞卜。元代表创始和善良,亨象征亨通顺利,利意味着和谐有利,贞表示坚守正道。

**象辞**:
```
象曰:天行健,君子以自强不息。
```

**白话译文**:
《象辞》说:天道刚健,运行不已。君子观此卦象,从而以天为法,自强不息。

**彖辞**:
```
彖曰:大哉乾元,万物资始,乃统天。云行雨施,品物流形。大明终始,六位时成,时乘六龙以御天。乾道变化,各正性命,保合大和,乃利贞。首出庶物,万国咸宁。
```

**爻辞详解**:

**初九**:
```
初九:潜龙勿用。
象曰:潜龙勿用,阳在下也。
```
白话:龙潜伏在水中,暂时不要有所作为。此时阳爻在最下位,力量尚未显现,应韬光养晦。

**九二**:
```
九二:见龙在田,利见大人。
象曰:见龙在田,德施普也。
```
白话:龙出现在田野上,利于会见贵人。此时已崭露头角,有利于得到大人物的提携和指导。

**九三**:
```
九三:君子终日乾乾,夕惕若,厉无咎。
象曰:终日乾乾,反复道也。
```
白话:君子整日自强不息,夜晚也保持警惕,虽处危境但无灾祸。应时刻勤勉谨慎,反复践行正道。

**九四**:
```
九四:或跃在渊,无咎。
象曰:或跃在渊,进无咎也。
```
白话:龙或跃起或退守深渊,都不会有灾祸。此时进退应审时度势,灵活应对。

**九五**:
```
九五:飞龙在天,利见大人。
象曰:飞龙在天,大人造也。
```
白话:龙飞翔在天空,利于出现大德之人。此为最吉位,居中得正,大有作为之时。

**上九**:
```
上九:亢龙有悔。
象曰:亢龙有悔,盈不可久也。
```
白话:龙飞得太高,会有悔恨。物极必反,盛极而衰,过刚则折。

**用九**:
```
用九:见群龙无首,吉。
象曰:用九天德,不可为首也。
```
白话:看到群龙都不以首领自居,吉祥。天德广大,不独占功劳。

**现代应用指导**:
- **事业**:大吉大利,春风得意,但需警惕盛极必衰,保持谦逊
- **财运**:财运亨通,投资顺利,但不可贪心过度
- **感情**:阳盛阴衰,需刚柔相济,男性易过刚,女性温柔者更宜
- **健康**:精力充沛,但需防止过劳,注意头部、神经系统

### 第2卦 坤卦(坤为地) ䷁

**卦象组成**:
- 上卦:坤☷(地)
- 下卦:坤☷(地)
- 六爻:上六、六五、六四、六三、六二、初六(六爻皆阴)

**卦辞**:
```
坤:元,亨,利牝马之贞。君子有攸往,先迷后得主,利。西南得朋,东北丧朋。安贞吉。
```

**白话译文**:
坤卦:大吉大利,像母马那样柔顺坚定最为吉利。君子前行,起初会迷失方向,后来得到主导就吉利。往西南方可得到朋友,往东北方会失去朋友。安守正道则吉祥。

**象辞**:
```
象曰:地势坤,君子以厚德载物。
```

**白话译文**:
《象辞》说:大地的形势坤厚和顺,君子观此卦象,应当效法大地,以深厚的德行承载万物。

**彖辞**:
```
彖曰:至哉坤元,万物资生,乃顺承天。坤厚载物,德合无疆。含弘光大,品物咸亨。牝马地类,行地无疆,柔顺利贞。君子攸行,先迷失道,后顺得常。西南得朋,乃与类行;东北丧朋,乃终有庆。安贞之吉,应地无疆。
```

**爻辞详解**:

**初六**:
```
初六:履霜,坚冰至。
象曰:履霜坚冰,阴始凝也。驯致其道,至坚冰也。
```
白话:踩到霜冻,坚冰即将来临。阴气初凝,渐渐发展就会形成坚冰,提醒人要防微杜渐。

**六二**:
```
六二:直,方,大,不习无不利。
象曰:六二之动,直以方也。不习无不利,地道光也。
```
白话:正直、方正、广大,不需要学习练习也无所不利。坤德光明,顺应自然就能成功。

**六三**:
```
六三:含章可贞。或从王事,无成有终。
象曰:含章可贞,以时发也。或从王事,知光大也。
```
白话:蕴含美德可守正道。如果辅佐君王,不求居功但能善终。美德适时展现,辅佐君王体现智慧。

**六四**:
```
六四:括囊,无咎无誉。
象曰:括囊无咎,慎不害也。
```
白话:扎紧口袋,谨言慎行,虽无功誉但也无灾祸。谨慎行事可避免伤害。

**六五**:
```
六五:黄裳,元吉。
象曰:黄裳元吉,文在中也。
```
白话:穿黄色下裳,大吉。黄色象征中正之德,美德在内,位居中正,大吉之象。

**上六**:
```
上六:龙战于野,其血玄黄。
象曰:龙战于野,其道穷也。
```
白话:龙在原野上争战,流出深黄色的血。阴气极盛,与阳气冲突,盛极则衰。

**用六**:
```
用六:利永贞。
象曰:用六永贞,以大终也。
```
白话:利于永远坚守正道。坤道以柔顺为本,永守正道可得善终。

**现代应用指导**:
- **事业**:宜谨慎守成,不宜急进,顺势而为,厚德载物
- **财运**:财运平稳,宜守不宜攻,稳扎稳打
- **感情**:柔顺包容,以退为进,女性得此卦尤为吉利
- **健康**:注意脾胃消化系统,保持平和心态

### 第3卦 屯卦(水雷屯) ䷂

**卦象组成**:
- 上卦:坎☵(水)
- 下卦:震☳(雷)
- 特征:水在雷上,云雷初动,万物始生

**卦辞**:
```
屯:元,亨,利,贞。勿用,有攸往,利建侯。
```

**白话译文**:
屯卦:大为亨通,利于坚守正道。不要轻举妄动,但利于建立诸侯,任用贤能。

**象辞**:
```
象曰:云雷,屯。君子以经纶。
```

**白话译文**:
《象辞》说:云与雷相合形成屯卦。君子观此卦象,应当治理国事,经营天下。

**爻辞详解**:

**初九**:
```
初九:磐桓,利居贞,利建侯。
象曰:虽磐桓,志行正也。以贵下贱,大得民也。
```
白话:徘徊不前,利于安居守正,利于建立封侯。虽然徘徊,但志向端正,能得民心。

**六二**:
```
六二:屯如邅如,乘马班如。匪寇婚媾,女子贞不字,十年乃字。
象曰:六二之难,乘刚也。十年乃字,反常也。
```
白话:困难重重如骑马盘旋不前。来者非寇而是求婚之人,女子坚守不嫁,十年后才出嫁。

**六三**:
```
六三:既鹿无虞,惟入于林中。君子几,不如舍,往吝。
象曰:既鹿无虞,以从禽也。君子舍之,往吝穷也。
```
白话:追逐鹿而无向导,只会迷失在森林中。君子应当审时度势,不如放弃,贸然前往必有困窘。

**现代应用指导**:
- **事业**:创业初期,艰难险阻,宜谨慎行事,不可冒进
- **财运**:初期困难,需耐心积累,不宜大规模投资
- **感情**:初期波折较多,需要时间考验,不宜急躁
- **健康**:注意肾脏、泌尿系统健康

### 第4卦 蒙卦(山水蒙) ䷃

**卦象组成**:
- 上卦:艮☶(山)
- 下卦:坎☵(水)
- 特征:山下有泉,蒙昧待启

**卦辞**:
```
蒙:亨。匪我求童蒙,童蒙求我。初筮告,再三渎,渎则不告。利贞。
```

**白话译文**:
蒙卦:亨通。不是我去求蒙昧的童子,而是童子来求教于我。初次占筮给予指教,再三轻慢地来问就不再告知。利于守正。

**象辞**:
```
象曰:山下出泉,蒙。君子以果行育德。
```

**白话译文**:
《象辞》说:山下流出泉水,这是蒙卦的象征。君子观此卦象,应当果断行事,培育德行。

**现代应用指导**:
- **事业**:处于学习成长期,应虚心求教,接受教育
- **财运**:不宜急于求成,先打好基础再谋发展
- **感情**:关系蒙昧不清,需要真诚沟通,建立信任
- **健康**:注意脾胃和四肢关节保养

### 阴阳爻说明

- **阳爻**: ⚊ (—) 用"九"表示,象征刚健、主动、阳刚
- **阴爻**: ⚋ (- -) 用"六"表示,象征柔顺、被动、阴柔

### 六爻位置

从下往上依次为:
1. 初爻(初九/初六):最底层,代表基础、开始
2. 二爻(九二/六二):下卦中位,代表臣位
3. 三爻(九三/六三):下卦上位,代表人位
4. 四爻(九四/六四):上卦下位,代表地位
5. 五爻(九五/六五):上卦中位,代表君位(最尊贵)
6. 上爻(上九/上六):最顶层,代表极限、终结

## 占卜方法说明

### 传统铜钱卜卦法

**所需工具**:三枚相同的铜钱(或硬币)

**步骤**:
1. 心中默念所问之事,诚心静气
2. 将三枚铜钱同时抛出,记录正反面
3. 重复6次,从下往上记录,形成完整卦象

**记录规则**:
- 三个正面:老阳(阳爻,变爻) 记作"O"
- 三个反面:老阴(阴爻,变爻) 记作"X"
- 两正一反:少阳(阳爻) 记作"—"
- 两反一正:少阴(阴爻) 记作"- -"

**变爻说明**:
- 老阳和老阴为变爻,会转化成相反的爻
- 变爻所在位置的爻辞最为重要
- 有变爻时需要看本卦和变卦两个卦象

### 示例

假设六次投掷结果为(从下往上):
1. 两正一反 → 少阳 ⚊
2. 三个反面 → 老阴 ⚋ (变爻,将变为阳爻)
3. 两反一正 → 少阴 ⚋
4. 两正一反 → 少阳 ⚊
5. 三个正面 → 老阳 ⚊ (变爻,将变为阴爻)
6. 两正一反 → 少阳 ⚊

**本卦**: 看第2爻(六二)和第5爻(九五)的爻辞
**变卦**: 将变爻转换后形成新卦,综合判断

## 附录:术语解释

### 基本术语

- **卦辞**:对整个卦象的总体判断和说明
- **爻辞**:对每一爻的具体说明
- **象辞**:《象传》对卦象和爻象的解释,分大象和小象
- **彖辞**:《彖传》对卦辞的详细阐释
- **文言**:专门解说乾坤两卦的文字

### 吉凶术语

- **元**:开始、创始、根本、大善
- **亨**:通达、顺利、亨通
- **利**:有利、适宜
- **贞**:正固、坚守正道、占问
- **吉**:吉祥、好
- **凶**:凶险、不好
- **咎**:过失、灾祸
- **悔**:后悔、遗憾
- **吝**:困难、艰涩
- **厉**:危险

### 卦象术语

- **中正**:爻位居中且阴阳得位
- **得位**:阳爻在阳位(1,3,5),阴爻在阴位(2,4,6)
- **失位**:阴阳爻位不对应
- **乘刚**:阴爻在阳爻之上
- **承柔**:阳爻在阴爻之下
- **应**:初爻与四爻、二爻与五爻、三爻与上爻相呼应
